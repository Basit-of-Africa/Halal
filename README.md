# Halal Heart Throb

Pious pickup lines for the modern Muslim. Built with Next.js, Tailwind CSS, ShadCN UI, and Genkit.

## 🚀 Launching on Vercel

This project is optimized for Vercel deployment. Follow these steps to go live:

1.  **Push to GitHub**: Push your local code to a new GitHub repository.
2.  **Connect to Vercel**:
    - Log in to your [Vercel Dashboard](https://vercel.com).
    - Click **"Add New..."** and then **"Project"**.
    - Import your GitHub repository.
3.  **Configure Environment Variables**:
    - During the setup, click on the **Environment Variables** section.
    - Add `GOOGLE_GENAI_API_KEY` and set its value to your Gemini API key (from [Google AI Studio](https://aistudio.google.com/)).
4.  **Deploy**: Click **"Deploy"**. Vercel will automatically detect the Next.js framework and build your app.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: ShadCN UI (Radix UI)
- **AI**: Genkit with Google Gemini
- **Icons**: Lucide React

## 📖 Features

- **20+ Pious Pickup Lines**: Categorized into Cheesy, Sincere, Spiritual, and Funny.
- **Favorites System**: Save your favorite lines to your browser's local storage.
- **Sharing**: One-tap copy to clipboard and native Web Share API integration.
- **Contextual Tips**: Understand the best way to deliver each line.
