# **App Name**: Halal Heart Throb

## Core Features:

- Flashcard Display: Display a single Halal pickup line on a flashcard.
- Context Reveal: Reveal a context tip for the displayed line with a tap or click.
- Flashcard Navigation: Enable users to navigate to the next or previous flashcard.
- Favorite Lines: Allow users to mark lines as favorites for easy access.

## Style Guidelines:

- Primary color: Rich purple (#6A4C93), suggesting royalty and respect.
- Background color: Desaturated light lavender (#E6E0EB), for a calming and neutral base.
- Accent color: Deep cream (#E3DAC9), for highlights and important interactive elements, creating a warm, inviting feel.
- Font pairing: 'Playfair' for headings and 'PT Sans' for body text. 'Playfair' is a modern sans-serif, while 'PT Sans' is a humanist sans-serif.
- Use elegant, minimalist icons to represent actions and categories, maintaining a clean and sophisticated look.
- Ensure a clean, card-based layout that is easy to navigate, emphasizing the simplicity of the content.
- Use subtle, smooth transitions between flashcards to enhance user experience.
